A modern, elegant beauty salon website inspired by Cristi Pascu Exclusive salon.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Beautiful, elegant design with smooth animations
- **Sections Included**:

  - About Us section

  - Statistics counter animation
  - Monthly offers/packages
  - Products section (placeholder)
  - Gallery section

  - Cookie consent banner

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Poppins)

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4af37;  /* Gold color */
    --secondary-color: #2c2c2c; /* Dark gray */
    /* ... */
}
```

### Content
- Edit text content directly in `index.html`
- Update contact information in the contact section
- Modify services, offers, and other content as needed

### Images
Replace the gallery placeholders with actual images:
- Add your images to an `images` folder
- Update the gallery items in `index.html` with `<img>` tags

### Form Submission
The contact form currently shows an alert. To make it functional:
1. Set up a backend service (e.g., PHP, Node.js)
2. Update the form submission handler in `script.js`
